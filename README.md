# 房地产帝国 - Vue3 版本

中国房地产2008-2028全行业真实流程模拟游戏

## 技术栈

- Vue 3.4 + TypeScript
- Vite 5
- TailwindCSS 3
- Pinia 2
- Vue Router 4

## 项目结构

```
/
├── src/
│   ├── main.ts                 # 应用入口
│   ├── App.vue                 # 根组件
│   ├── router/                 # 路由配置
│   ├── stores/                 # Pinia状态管理
│   ├── views/                  # 页面组件
│   ├── components/             # 通用组件
│   ├── types/                  # TypeScript类型定义
│   └── styles/                 # 样式文件
├── pmc/                        # 原始项目（保留）
└── ...
```

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```
